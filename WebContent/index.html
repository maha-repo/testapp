This listing installs support into Eclipse for Pivotal Cloud Foundry, the industry’s first open platform as a service. A Java 7 Execution Environment is required. Run your Spring, Java, and Grails applications. Spring deployment to Cloud Foundry now also includes Spring Boot. Deploy from your IDE or command line. Introducing Cloud Foundry Cloud Foundry is the open platform as a service project by Pivotal. It can support multiple frameworks, multiple cloud providers, and multiple application services all on a cloud scale platform. Developer Productivity — Build applications with your choice of high productivity frameworks and application services. Open System — Choose your framework, choose your cloud, choose your application services. Faster Delivery — Shorten the time it takes to take your application from concept, to code, to the cloud using an open platform as a service.
